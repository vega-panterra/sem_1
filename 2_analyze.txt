Анализ структуры страницы сайта https://ru.wikipedia.org/

1. Главная страница, на которой указаны название сайта (шапка), описание о нас, возможности кнопки создания статей.

2. Далее блок входа на сайт с полем для ввода данных для поиска на сайте и дополнительные возможности.

3. Основная часть страницы - контент сайта с блоками информации - "Избранная статья"
                                                                  "Из новых материалов"
                                                                  "Хорошая статья"
                                                                  "Текущие события"
                                                                  "В этот день"
                                                                  "Последний избранный список"
                                                                  "Совместная работа недели"
                                                                  "Изображение дня"
Данные блоки информации со вложенными ссылками для перехода.

4. Слева колонка с меню. Здесь есть содержание, статьи, сведения о правилах поведения, сообщество, форум и различные инструменты.
Также здесь есть возможность выбора языка.

5. В подвале страницы размещена информация о политике конфиденциональности, лицензии, связь с сайтом, мобильная версия и другие разделы.
